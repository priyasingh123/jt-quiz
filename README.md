# About the App
This App is created using reactJs.

## Components of App
It starts with entry page which asks for signing in, which is mandatory field.
After clicking on Submit, quiz starts with 15 questions and timer 30 mins.
Component Timer => starts as soon as first question is rendered.
Navbar component => keeps track on which question we are on. If question is visited, question on navbar is marked as yellow, if answered it is marked as green.
Submit Button is provided to submit quiz anytime before 30 mins.
On submission of quiz either automatically(timer expires) or willfully, results page is displayed. 
Report component => shows score, performance of user and correct answer as well as user's response to questions.
